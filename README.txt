Instructions d'installation et d'utilisation
-----------------------------------------

1. **Désactiver l'antivirus :**
   Avant de lancer l'installation, il est nécessaire de désactiver temporairement votre logiciel antivirus pour éviter toute interférence avec le programme.

2. **Lancer le fichier setup.bat :**
   Une fois l'antivirus désactivé, localisez le fichier `setup.bat` dans le dossier d'installation du programme. Faites un clic droit dessus et sélectionnez "Exécuter en tant qu'administrateur". Suivez les instructions à l'écran pour compléter l'installation.

3. **Installation terminée :**
   Une fois l'installation terminée, vous pouvez réactiver votre antivirus. Le programme est maintenant installé et prêt à être utilisé.
